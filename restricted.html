jQuery Restricted Plugin

Restricted is a jQuery plugin that restricts numerical, alphabetical, and special characters.

Restricted Plugin Dependencies

To use restricted, it is recommended that you use jQuery version 1.8+. Reference the jQuery library followed the restricted JavaScript file:

<!-- load jQuery library -->
<script src="/common/js/libs/jquery-1.8.min.js"></script>

<!-- load restricted plugin -->
<script src="jquery-restricted.js"></script>

Now you are ready to use restricted! Below is some example code on the ways you can use it.

Restricted Plugin Options

Options via HTML5 Data Attribute

Data-Attribute Value Description
data-type
  • alphabetical
  • numeric
  • alphanumeric
 Restricts field to specified character type.

NOTE: alphanumeric is the default type and is therefore optional and does not need to be set.

Example

<input type="text" class="restricted" data-type="data-type">

Options via JavaScript Property

Property Value Description
type
  • alphabetical
  • numeric
  • alphanumeric
 Restricts field to specified character type.

NOTE: alphanumeric is the default type and is therefore optional and does not need to be set.

Example

$('.restricted').restricted({ type: type });
